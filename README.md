# de-Loyola-Guilds

This project is a website for the de-Loyola-Guilds group, created to provide information and resources related to the guilds at Loyola University. The website aims to enhance community engagement and showcase guild activities.

## Live Website

You can view the live website at [de-Loyola-Guilds on GitHub Pages](https://your-username.github.io/de-Loyola-Guilds/).

## Features
- Information about different guilds
- Events calendar
- Registration forms
- Contact information

## Getting Started

To get a local copy up and running follow these simple steps:

1. Clone the repo
   ```sh
   git clone https://github.com/KuraKat/de-Loyola-Guilds.git
   ```
2. Install the required dependencies
3. Run the project locally
